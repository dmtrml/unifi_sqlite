# Release Notes — 2024-12-15

## Highlights
- Добавлен `roadmap.md` с текущим состоянием и детальными планами по Dashboard, импорту, транзакциям, счетам, категориям, бюджетам, recurring и отчётам.
- Отчётный донат-чарт (`CategorySpendingChart`) показывает дочерние категории в тултипе (сумма + % от родителя и общей суммы), а сегмент «Other» больше не кликабелен.
- `CategoryBreakdownChart` получил цветные иконки подкатегорий, обновлённые прогресс-бары (учитывают собственные расходы родителя) и корректный фильтр `Uncategorized`.
- Страница `Categories` отображает иконки подкатегорий (десктоп и мобайл), вместо одноцветных точек.
- Фильтры и API теперь поддерживают `includeUncategorized` + `transactionType`, поэтому детальная модалка отчётов показывает только нужные транзакции без категорий.

## Technical
- `useTransactions`, `/api/transactions`, `TransactionsRepository` и отчётные эндпоинты расширены новыми параметрами.
- Добавлены вспомогательные утилиты для отображения дерева категорий в отчётах и UI.
- UI-полировки: прогресс-бары, тултипы, списки категорий и отображение детей синхронизированы по иконкам/цветам.

## Next Steps
См. `roadmap.md` — в нём перечислены задачи по Dashboard, импорту, транзакциям, счетам, бюджетам, recurring и отчётам (включая фиксы для Income vs. Expense).
