# **App Name**: BudgetWise

## Core Features:

- Expense Recording: Record new expenses, assigning them to predefined categories for tracking.
- Budget Setting: Set monthly budget goals for different categories (e.g., food, rent, entertainment).
- Interactive Dashboard: Display total spending, remaining budget, and category breakdowns in an easily digestible format.
- Visual Charts: Generate visual charts (e.g., bar graphs, pie charts) of spending trends over time.
- Recurring Expenses: Automatically record monthly, bi-weekly, or weekly expenses after the initial input.

## Style Guidelines:

- Primary color: Teal (#008080) for a sense of calm and financial stability.
- Background color: Light gray (#D3D3D3) to provide a clean, professional backdrop.
- Accent color: Dark blue (#00008B) to highlight key information and call-to-action elements.
- Font pairing: 'Inter' for body text, a grotesque-style sans-serif for a modern and neutral look, and 'Space Grotesk' sans-serif for headers.
- Use clear and intuitive icons to represent expense categories and navigation elements.
- Clean and structured layout with a focus on readability and easy navigation. Use clear visual hierarchy to guide the user through the app.
- Subtle animations and transitions to provide a smooth and engaging user experience when loading new content.
- The design should be minimalist, intuitive, and visually clean, with a responsive interface that adapts seamlessly to different devices (mobile, tablet, and desktop).
